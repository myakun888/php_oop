# pengenalan object oriented programming

object oriented programming adalah suatu cara pandang programming atau sudut pandang programming berdasar object
ada banyak sudut pandang dalam pemrograman
functional programing , oop dll

saat ini oop adalah sudut pandang yang populer , atau banyak di gunakan oleh programmer saat ini

dalam oop , ada beberapa istilah yang perlu kita mengerti diantaranya `object` dan `class`;

                                                                
## object
object adalah data yang berisi field/atributes/propertis, function / method , dan behavior lain

##class
class adalah cetakan atau blueprint untuk membuat object

satu class bisa untuk membuat banyak object
satu object hanya bisa memiliki satu class
