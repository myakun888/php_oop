# class
clas adalah blue print atau cetakan untuk membuat banyak object
seperti sudah di jelaskan di materi sebelumnya



## membuat class

cukup dengan kata kunci `class`
seperti berikut

```
class nama_class{

}

 ```


ini adalah class dengan nama person
```php
class Person 
{

}

```

saat membuat nama class di rekomendasi kan dengan syle pascal case
dimana selalu di awali dengan huruf besar , jika dua kata huruf kalimat selanjutnya juga huruf besar

```php
class KitabSuci {

    
}

```
